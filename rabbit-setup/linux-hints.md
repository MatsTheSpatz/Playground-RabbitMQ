# Hints for working in Ubuntu

Do everything as **root**:
``` sudo -s ```

---
Keyboard shortcut to **open terminal**:
``` ctrl + alt + T ```

---
**Nano Editor:**

Copy a line: 
````
Ctrl + K   // -> cut lline
Ctrl + U   // -> insert line
````

Select text: 
````
Alt + A   // -> starts selection - then move with cursor
// then use Ctrl + K to cut and Ctrl + U to paste.
````
